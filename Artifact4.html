DIY Color Mixer

This artifact was a DIY Color Mixer, created to help painters utilize the colors they had to get the different colors they wanted. This was a solo project I built using HTML, CSS, & JavaScript; connecting to AI to give users the answers they were after.

I learned how to code for HTML & CSS; how to set up a server & make prompts; and how to use code to build the project I wanted to make.

I really enjoyed creating the background animation for my website. Also seeing the prompt be used to actually create the paints needed by users was rather rewarding.
