A Knight at the Magician's Manor

A Knight at the Magician's Manor was a AR team project about a Knight venturing into a mansion to escape the storm only to be trapped inside by the Magician who lived within. Utilizing a set of cards scanned by the Player's camera could give the Knight abilities to explore the mansion & defeat the Magician to escape.

I learned that it is very important to work hard to get everything done sooner as bugs and problems will appear & can then complete them before the project ends. Making certain everyone turns in their work is very important, and work required for other members should be given when requested.

That and building the cards required for the game allowed me to practice and improve my character art.

What is Light?

What is Light was a point & click adventure team project, created to teach about the meaning of light, and what it is. The main character is stuck inside of a building and pushed to take the two paths in front of them: light is a wave, or light is a proton. As the player journeys through each path they go through many of the same rooms that change between both sides. They meet a friend who helps the player continue on, arguing scientists on what light, a being of light and a mysterious man who scorns the player.

I learned that scope is very important when building a project. If there is a problem that is causing quite a bit of trouble, handle it sooner rather than later. Code is a weak point for me that I need to improve on.

I really enjoyed drawing the two scientists, building their characters & interactions with the player.

DIY Color Mixer

This artifact was a DIY Color Mixer, created to help painters utilize the colors they do have to get the colors wanted. This was a solo project I built using HTML, CSS, & JavaScript, connecting to AI to give users the answers they were after.

I learned a lot of coding for HTML & CSS, how to set up a server & make prompts, and how to use code to build the project I want to make.

I really enjoyed creating the background animation for my website. Also seeing the prompt be used to actually create the paints needed by users was rather rewarding.
